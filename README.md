# CA-Net Video Segmentation App

A Streamlit-based video segmentation application using CA-DeepLab architecture with Channel Attention (CA) blocks.

## Features

- Video Upload
- CA-Net Segmentation
- Blend Overlay Output
- Side-by-Side Preview
- Download Processed Videos

## Tech Stack

- Python
- PyTorch
- Streamlit
- OpenCV
- DeepLabV3 ResNet50

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
